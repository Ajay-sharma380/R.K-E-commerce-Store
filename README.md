# R.K-E-commerce-Store
he R.K Store Report 2022 is a fully interactive and visually rich Excel Dashboard Project designed to analyze the complete sales and order performance of R.K Store throughout the year 2022. This dashboard offers a comprehensive understanding of business performance using dynamic charts, timeline slicers, category filters, and advanced Excel analytics techniques. It reflects real-world reporting standards and is ideal for data analytics learning, practice, and portfolio building.

This project includes detailed analysis across multiple business dimensions such as sales channels, product categories, customer demographics, order statuses, and geographic distribution. Through an intuitive dashboard layout, users can quickly explore patterns, compare metrics, and draw valuable insights for decision-making.

One of the main highlights of this project is the Order Status Analysis, which clearly shows Delivered, Returned, Cancelled, and Refunded orders, enabling better understanding of operational efficiency. Along with this, the Channel Orders Visualization highlights the performance of top marketplaces like Amazon, Myntra, Flipkart, Ajio, Meesho, Nalli, and others, revealing which platforms generated the highest order volume.

The dashboard also features an Orders vs. Sales Monthly Trend, allowing users to see the relationship between total orders and revenue across all twelve months. This helps identify seasonal demand, peak months, and revenue consistency. Another key visualization is the Top 5 States Sales Chart, showcasing the highest revenue-generating regions such as Maharashtra, Karnataka, Uttar Pradesh, Telangana, and Tamil Nadu.

To understand customer behavior, the project includes two demographic visuals: Age vs Gender Orders and Men vs Women Sales Contribution, which offer insights into which age groups and genders contribute the most to overall sales.
